# SO.CLOTHING — React + Vite + Tailwind + GSAP

A bold, cinematic streetwear demo site. 11 pages, GSAP ScrollTrigger parallax, light modern theme.

## Quick start

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Stack
- React 18 (JavaScript, no TypeScript)
- Vite 5
- Tailwind CSS 3
- GSAP 3 + ScrollTrigger (parallax, pinned sections, scroll-driven reveals)
- React Router 6
- Sonner (toasts), Zod (form validation), Lucide icons

## Pages
Home · Shop · Product · Cart · Checkout · Login · Register · Account · About · Contact · Lookbook
